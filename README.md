Issue Reporting Platform for Rural Schools

A role-based web application built using Django to digitize and streamline the issue reporting and grievance management process in rural educational institutions.

Key Features

Secure authentication for Students, Staff, and Admin

Role-Based Access Control (RBAC)

Issue submission with category and subject tagging

Real-time status tracking (Pending, In Progress, Solved)

Admin dashboard for user and complaint management

PDF export of complaint details

Search and filter functionality

Data aggregation and reporting dashboard

Tech Stack

Backend: Django (Python)

Frontend: HTML, CSS, JavaScript

Database: PostgreSQL

Reporting: ReportLab (PDF generation)

Version Control: Git

Objective

To improve transparency, accountability, and operational efficiency in rural schools by replacing manual complaint handling systems with a scalable digital platform.
